6. Проверка на нормальность

Задание 6.2

import numpy as np
import scipy.stats as stats

# задаём уровень значимости
alpha = 0.05 

# загружаем данные
data = [12,  8,  9, 10, 11, 12,  7, 12,  2, 10,  3,  9, 18, 4, 11, 13, 13, 17, 14,  3, 10,  2,  2,  7, 14,  8, 20,  5,  7, 10]

# проводим тест Шапиро — Уилка
result = stats.shapiro(data)
p = result[1]

print('p-value = {:.3f}'.format(p))

# интерпретируем результат
if p <= alpha:
    print('Распределение не нормальное')
else:
    print('Распределение нормальное')

# p-value = 0.414
# Распределение нормальное

7. Параметрические тесты

Задание 7.3

Необходимо провести одновыборочный t-test. Это делается с помощью функции scipy.stats.ttest_1samp().

Сформулируем гипотезы для вопроса компании:

    • Нулевая гипотеза:среднее время доставки больше или равно 40 минутам.

    • Альтернативная гипотеза: среднее время меньше 40 минут.

Необходимо использовать параметр теста alternative='less'.

Строка запуска теста (data — список выборочных значений):

stats.ttest_1samp(data, popmean=40, alternative='less') 
# p-value = 0.018

Задание 7.4

Необходимо провести тест ANOVA. Это делается с помощью функции scipy.stats.f_oneway().

Сформулируем гипотезы для вопроса компании:

    • Нулевая гипотеза: среднее время доставки в Северный (), Южный () и Восточный () районы одинаковое.

    • Альтернативная гипотеза: среднее время доставки в Северный (), Южный () и Восточный () районы не одинаковое.

Строка запуска теста (data_north, data_south, data_east — выборочные данные):

stats.f_oneway(data_north, data_south, data_east)
# p-value = 0.002

Задание 7.5

Необходимо провести парный t-test. Это делается с помощью функции scipy.stats.ttest_rel().

Сформулируем гипотезы для вопроса компании:

    • Нулевая гипотеза: среднее время доставки до обучения () меньше или равно среднему времени доставки после обучения ().

    • Альтернативная гипотеза: среднее время доставки до обучения () больше среднего времени доставки после обучения ().

Необходимо использовать параметр теста alternative='greater'.

Строка запуска теста (data_before, data_after — выборочные данные):

stats.ttest_rel(data_before, data_after, alternative = 'greater')
# p-value = 0.213
